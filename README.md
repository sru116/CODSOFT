## Objective
To classify Iris flowers into Setosa, Versicolor, and Virginica species using Machine Learning.

- Pandas
- Matplotlib
- Scikit-Learn

## Algorithm
Decision Tree Classifier

## Accuracy
100%

## Outcome
Successfully classified Iris flower species based on sepal and petal measurements.
